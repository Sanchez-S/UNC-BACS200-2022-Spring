### Bootstrap Navs
* Use class "nav" on list
* Use class "nav-item" on each list item
* Use class "nav-link" on each link element

```
    <ul class="nav">
      <li class="nav-item">
        <a class="nav-link active" href="#">Home</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">About</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Profile</a>
      </li>
    </ul>
```
