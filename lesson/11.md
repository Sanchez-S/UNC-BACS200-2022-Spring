### Basic Selectors
* Element (\<element type\>)
* Class (.\<***user-defined*** class name\>)
* ID (#\<***user-defined*** ID\>)
* Pseudo-class (:\<pseudo-class name\>)
* Descendant (\<parent selector\> \<descendant selector\>)

### Basic Selectors

| Selector     | Syntax                                      |
| :----------- | :------------------------------------------ |
| Element      | \<element type\>                            |
| Class        | \<***user-defined*** class name\>           |
| ID           | #\<***user-defined*** ID\>                  |
| Pseudo-class | :\<pseudo-class name\>                      |
| Descendant   | \<parent selector\> \<descendant selector\> |
